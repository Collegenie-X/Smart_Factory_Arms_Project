# OLED 디스플레이 (SSD1306) 제어 예제

## 📌 개요
128x64 OLED 디스플레이를 사용하여 텍스트, 그래픽, 애니메이션을 표시하는 예제입니다.

## 🔧 하드웨어
- **0.96" OLED (SSD1306)**
- **해상도**: 128x64 픽셀
- **통신**: I2C
- **색상**: 단색 (흰색/파랑)

## 📍 핀 연결
```
Arduino UNO
├─ A4  → SDA (I2C 데이터)
├─ A5  → SCL (I2C 클럭)
├─ 5V  → VCC
└─ GND → GND

I2C 주소: 0x3C (일반적)
```

## 🎯 기능
1. **텍스트 표시**: 여러 줄 자동 줄바꿈
2. **그래프**: 실시간 센서 데이터 시각화
3. **아이콘**: 온도, 습도, 체크, 경고 아이콘
4. **차트**: 막대 그래프
5. **스크롤 텍스트**: 좌우 스크롤
6. **센서 대시보드**: 다중 센서 데이터 표시
7. **애니메이션**: 원, 사각형 애니메이션

## 💻 Serial 명령어
| 명령어 | 기능 |
|--------|------|
| `text:메시지` | 텍스트 표시 |
| `graph` | 실시간 그래프 |
| `icon` | 아이콘 표시 |
| `chart` | 차트 표시 |
| `scroll` | 스크롤 텍스트 |
| `sensor` | 센서 대시보드 |
| `anim` | 애니메이션 |
| `clear` | 화면 지우기 |

## 📊 좌표 시스템
```
(0,0)                    (127,0)
  ┌───────────────────────┐
  │                       │
  │     128 x 64          │
  │                       │
  └───────────────────────┘
(0,63)                  (127,63)
```

## 📝 주요 함수
- `displayText()`: 텍스트 표시 (자동 줄바꿈)
- `displayGraph()`: 실시간 그래프
- `displayIcons()`: 아이콘 그리기
- `displayChart()`: 막대 그래프
- `scrollText()`: 스크롤 텍스트
- `sensorDashboard()`: 센서 대시보드
- `playAnimation()`: 애니메이션

## 💡 OLED vs LCD

| 항목 | OLED | LCD |
|------|------|-----|
| 해상도 | 128x64 픽셀 | 16x2 문자 |
| 그래픽 | 가능 | 제한적 |
| 백라이트 | 불필요 | 필요 |
| 시야각 | 넓음 | 좁음 |
| 가격 | 중간 | 저렴 |
| 소비 전력 | 낮음 | 높음 |
