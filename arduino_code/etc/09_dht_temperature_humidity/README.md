# DHT11/DHT22 온습도 센서 측정 예제

## 📌 개요
DHT11 또는 DHT22 센서를 사용하여 온도와 습도를 측정하는 예제입니다.

## 🔧 하드웨어
- **DHT11** (정확도: ±2°C, ±5%) 또는
- **DHT22** (정확도: ±0.5°C, ±2%)
- **LED** 3개 (온도 범위 표시)

## 📍 핀 연결
```
Arduino UNO
├─ D2  → DHT 데이터 핀
├─ D3  → 파랑 LED (낮은 온도)
├─ D4  → 초록 LED (적정 온도)
└─ D5  → 빨강 LED (높은 온도)
```

## 🎯 측정 항목
1. **온도** (°C)
2. **습도** (%)
3. **불쾌지수** (DI)
4. **체감온도** (Heat Index)
5. **이슬점** (Dew Point)

## 📊 온도 범위
| 온도 | 상태 | LED |
|------|------|-----|
| < 20°C | 춥다 | 파랑 |
| 20~26°C | 적정 | 초록 |
| 26~30°C | 따뜻함 | 빨강 |
| > 30°C | 더움 | 빨강 깜빡임 |

## 📊 습도 범위
| 습도 | 상태 |
|------|------|
| < 30% | 건조 |
| 30~70% | 적정 |
| > 70% | 습함 |

## 💻 불쾌지수 계산
```cpp
DI = 0.81 × 온도 + 0.01 × 습도 × (0.99 × 온도 - 14.3) + 46.3

< 68: 쾌적
68~75: 보통
75~80: 약간 불쾌
80~85: 불쾌
> 85: 매우 불쾌
```

## 📝 주요 함수
- `readSensor()`: 온습도 읽기
- `calculateDiscomfortIndex()`: 불쾌지수 계산
- `calculateHeatIndex()`: 체감온도 계산
- `calculateDewPoint()`: 이슬점 계산

## 📚 필수 라이브러리
```
DHT sensor library (Adafruit)
```
