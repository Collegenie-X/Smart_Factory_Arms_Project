# LCD1602 고급 기능 예제

## 📌 개요
LCD1602 I2C의 고급 기능을 활용하여 커스텀 문자, 진행률 바, 메뉴 시스템 등을 구현하는 예제입니다.

## 🔧 하드웨어
- **LCD1602 I2C 모듈**
- **버튼** 3개 (위, 아래, 선택) - 옵션

## 📍 핀 연결
```
Arduino UNO
├─ A4  → SDA (I2C 데이터)
├─ A5  → SCL (I2C 클럭)
├─ D2  → 버튼 UP (옵션)
├─ D3  → 버튼 DOWN (옵션)
└─ D4  → 버튼 SELECT (옵션)
```

## 🎯 기능
1. **커스텀 문자**: 하트, 벨, 온도계, 물방울, 체크, 스피커
2. **스크롤 텍스트**: 긴 메시지 스크롤
3. **진행률 바**: 로딩 바 표시
4. **애니메이션**: 하트 깜빡임, 스피너
5. **메뉴 시스템**: 상하 이동, 선택
6. **센서 데이터**: 온습도 표시 (아이콘 포함)
7. **로딩 화면**: 퍼센트 표시

## 💻 Serial 명령어
| 명령어 | 기능 |
|--------|------|
| `custom` | 커스텀 문자 표시 |
| `scroll` | 스크롤 텍스트 |
| `progress` | 진행률 바 |
| `anim` | 애니메이션 |
| `menu` | 메뉴 시스템 |
| `sensor` | 센서 데이터 |
| `loading` | 로딩 화면 |

## 📊 커스텀 문자 생성
```cpp
// 5x8 픽셀 패턴 정의
byte heart[8] = {
  0b00000,  // 빈 줄
  0b01010,  // ⚫ ⚫
  0b11111,  // ⚫⚫⚫⚫⚫
  0b11111,  // ⚫⚫⚫⚫⚫
  0b11111,  // ⚫⚫⚫⚫⚫
  0b01110,  //  ⚫⚫⚫
  0b00100,  //   ⚫
  0b00000   // 빈 줄
};

// 문자 등록 (0~7번)
lcd.createChar(0, heart);

// 문자 출력
lcd.write(0);
```

## 📝 주요 함수
- `showCustomCharacters()`: 커스텀 문자 표시
- `scrollText()`: 스크롤 텍스트
- `showProgressBar()`: 진행률 바 (0~100%)
- `playAnimation()`: 애니메이션 효과
- `showMenu()`: 메뉴 시스템
- `showSensorData()`: 센서 데이터 (아이콘)
- `showLoadingScreen()`: 로딩 화면

## 🎨 진행률 바 구현
```cpp
// 16칸 진행률 바
for (int i = 0; i < 16; i++) {
  if (i < filled) {
    lcd.write(255);  // ■
  } else {
    lcd.print(" ");  // 빈칸
  }
}
```

## 🔍 응용 분야
- 메뉴 시스템
- 로딩 바
- 센서 모니터링 (아이콘)
- 상태 표시
- 사용자 인터페이스

## 💡 커스텀 문자 활용

### 1. 센서 아이콘
- 온도계: 온도 표시 시
- 물방울: 습도 표시 시
- 하트: 심박수 표시 시

### 2. 상태 아이콘
- 체크 마크: 성공
- X 표시: 실패
- 벨: 알림

### 3. 애니메이션
- 하트 깜빡임: 진행 중
- 스피너: 로딩 중
- 화살표: 메뉴 선택

## 📚 LCD 특수 문자
| 코드 | 문자 | 의미 |
|------|------|------|
| 0~7 | 커스텀 | 사용자 정의 |
| 223 | ° | 도(degree) |
| 255 | ■ | 채움 블록 |

## 🎓 16x2 제약사항
- 최대 8개 커스텀 문자
- 2줄 16자 제한
- 그래픽 불가 (문자만)
- 색상 불가 (단색)

## 💡 팁
- 커스텀 문자를 활용하여 표현력 향상
- 진행률 바로 사용자 피드백 제공
- 스크롤로 긴 메시지 표시
- 메뉴 시스템으로 사용자 인터페이스 구축
