# 사운드 + 진동 + 기울기 센서 통합 예제

## 📌 개요
사운드, 진동, 기울기 센서를 통합하여 다양한 이벤트를 감지하는 예제입니다.

## 🔧 하드웨어
- **사운드 센서** (마이크 모듈)
- **진동 센서** (SW-420)
- **기울기 센서** (Tilt Switch)
- **LED** 3개
- **부저** 1개

## 📍 핀 연결
```
Arduino UNO
├─ A0  → 사운드 센서 (아날로그)
├─ D7  → 진동 센서 (디지털)
├─ D8  → 기울기 센서 (디지털, 풀업)
├─ D2  → 사운드 감지 LED
├─ D3  → 진동 감지 LED
├─ D4  → 기울기 감지 LED
└─ D5  → 부저
```

## 🎯 센서별 기능

### 1. 사운드 센서
- **타입**: 아날로그 (0~1023)
- **감지**: 소음 레벨
- **임계값**: 300 (보통), 600 (시끄러움)

### 2. 진동 센서
- **타입**: 디지털 (LOW = 감지)
- **감지**: 진동/충격
- **용도**: 도난 방지, 지진 감지

### 3. 기울기 센서
- **타입**: 디지털 (LOW = 기울어짐)
- **감지**: 기울기 변화
- **용도**: 전도 방지, 각도 감지

## 📊 소음 레벨
| 값 | 레벨 | LED | 부저 |
|------|------|-----|------|
| 0~300 | 조용함 | OFF | OFF |
| 300~600 | 보통 | ON | OFF |
| > 600 | 시끄러움 | ON | ON |

## 📝 주요 함수
- `readSensors()`: 모든 센서 읽기
- `controlIndicators()`: LED 및 부저 제어
- `estimateDecibel()`: 데시벨 추정
- `countVibrations()`: 진동 횟수 카운트
- `monitorNoise()`: 소음 모니터링

## 💻 데시벨 추정

```cpp
// 간단한 로그 스케일 근사

```

## 🔍 응용 분야
- 보안 시스템 (진동 + 사운드)
- 소음 모니터링
- 전도 방지 알람 (기울기)
- 지진 감지기 (진동)

## 🎓 센서 조합 활용
| 조합 | 용도 |
|------|------|
| 사운드 + 진동 | 침입 감지 |
| 진동 + 기울기 | 전도 방지 |
| 사운드 + 기울기 | 도난 방지 |
