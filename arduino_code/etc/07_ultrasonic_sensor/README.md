# 초음파 센서 (HC-SR04) 거리 측정 예제

## 📌 개요
초음파 센서를 사용하여 정밀한 거리를 측정하는 예제입니다. 주차 센서, 장애물 회피 등에 활용됩니다.

## 🔧 하드웨어
- **HC-SR04 초음파 센서**
- **LED** 3개 (거리 단계 표시)
- **부저** 1개 (근접 경고)

## 📍 핀 연결
```
Arduino UNO
├─ D7  → TRIG (초음파 송신)
├─ D6  → ECHO (초음파 수신)
├─ D2  → 초록 LED (멀리, > 30cm)
├─ D3  → 노랑 LED (중간, 10~30cm)
├─ D4  → 빨강 LED (가까이, < 10cm)
└─ D5  → 부저 (근접 경고)
```

## 🎯 동작 원리
1. **초음파 송신**: TRIG 핀에 10μs 펄스 전송
2. **초음파 수신**: ECHO 핀에서 반사파 수신
3. **거리 계산**: 시간 × 음속(340m/s) ÷ 2
4. **LED 표시**: 거리에 따라 LED 색상 변경

## 📊 거리 범위
| 거리 | LED | 부저 |
|------|-----|------|
| < 10cm | 빨강 | ON (긴급) |
| 10~30cm | 노랑 | OFF |
| 30~100cm | 초록 | OFF |
| > 100cm | 모두 OFF | OFF |

## 💻 측정 공식
```cpp
거리(cm) = (시간(μs) × 0.034) / 2
```

## 📝 주요 함수
- `measureDistance()`: 거리 측정 (cm)
- `controlIndicators()`: LED 및 부저 제어
- `measureAverageDistance()`: 평균 거리 (노이즈 제거)
