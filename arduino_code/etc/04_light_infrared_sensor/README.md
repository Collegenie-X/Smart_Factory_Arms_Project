# 조도센서 + 적외선센서 통합 예제

## 📌 개요
조도센서(CdS)와 적외선센서를 사용한 자동 조명 제어 시스템 예제입니다.

## 🔧 하드웨어
- **조도센서** (CdS 포토레지스터)
- **적외선 장애물 센서**
- **LED** 2개 (조도용, 감지용)
- **부저** 1개

## 📍 핀 연결
```
Arduino UNO
├─ A0  → 조도센서 (아날로그)
├─ A1  → 적외선센서 (아날로그)
├─ D3  → 조도 연동 LED
├─ D4  → 감지 LED
└─ D5  → 부저
```

## 🎯 동작 원리
### 조도센서
- **어두움 감지**: 값 < 300
- **자동 조명**: LED 자동 ON/OFF
- **밝기 레벨**: 매우 어두움 / 어두움 / 보통 / 밝음

### 적외선센서
- **물체 감지**: 값 < 512
- **알림**: LED + 부저
- **거리 추정**: 매우 가까움 / 가까움 / 보통 / 멀리

## 📊 센서 값 범위
| 센서 | 범위 | 의미 |
|------|------|------|
| 조도센서 | 0~300 | 매우 어두움 |
| 조도센서 | 300~500 | 어두움 |
| 조도센서 | 500~700 | 보통 |
| 조도센서 | 700~1023 | 밝음 |
| 적외선센서 | 0~200 | 매우 가까움 (< 5cm) |
| 적외선센서 | 200~400 | 가까움 (5~10cm) |
| 적외선센서 | 400~600 | 보통 (10~20cm) |
| 적외선센서 | 600~1023 | 멀리 (> 20cm) |

## 📝 주요 함수
- `readSensors()`: 센서 값 읽기
- `handleLightSensor()`: 조도센서 처리 (자동 조명)
- `handleIRSensor()`: 적외선센서 처리 (물체 감지)
- `playDetectionSound()`: 감지 알림음
- `printStatus()`: 상태 출력

## 🔧 임계값 조정
```cpp
#define LIGHT_THRESHOLD  300   // 조도 임계값
#define IR_THRESHOLD     512   // 적외선 임계값
```
