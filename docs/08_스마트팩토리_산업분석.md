### 1.2 스마트 팩토리 vs 전통 공장

| 구분 | 전통 공장 | 스마트 팩토리 |
|------|----------|--------------|
| **생산 방식** | 대량 생산 | 맞춤형 생산 |
| **제어** | 수동/반자동 | 완전 자동/AI 제어 |
| **품질 관리** | 사후 검사 | 실시간 모니터링 |
| **유지보수** | 고장 후 수리 | 예측 정비 |
| **데이터** | 부분적 수집 | 전체 수집 + 분석 |
| **의사결정** | 관리자 | AI + 관리자 |
| **근로자 역할** | 단순 작업 | 고급 기술 + 관리 |
| **생산성** | 기준 100% | 150-300% |
| **불량률** | 3-5% | 0.5-1% |

## 2. 전문 용어 사전 (용어집)

### 2.1 제조 시스템 용어

| 용어 | 영문 | 정의 | 적용 사례 |
|------|------|------|----------|
| **MES** | Manufacturing Execution System | 제조 실행 시스템: 생산 현장의 작업 지시, 실시간 모니터링, 품질 관리를 담당하는 시스템 | 테슬라 공장 전체 라인 제어 |
| **ERP** | Enterprise Resource Planning | 전사적 자원 관리: 구매, 재고, 회계, 인사 등 기업 전체 자원을 통합 관리 | SAP, Oracle 시스템 |
| **PLM** | Product Lifecycle Management | 제품 수명 주기 관리: 설계부터 폐기까지 제품의 전 생애 관리 | 현대차 신차 개발 프로세스 |
| **SCADA** | Supervisory Control And Data Acquisition | 감시 제어 데이터 수집: 설비의 실시간 상태 감시 및 제어 | 발전소, 정유 공장 |
| **PLC** | Programmable Logic Controller | 프로그래머블 로직 컨트롤러: 산업 자동화를 위한 디지털 컴퓨터 | 컨베이어 벨트 제어 |
| **HMI** | Human-Machine Interface | 사람-기계 인터페이스: 작업자가 설비를 제어하는 화면 | 터치스크린 제어 패널 |
| **OEE** | Overall Equipment Effectiveness | 종합 설비 효율: 가동률 × 성능률 × 양품률 | 테슬라 95% OEE 목표 |
| **JIT** | Just-In-Time | 적시 생산: 필요한 것을 필요한 때에 필요한 만큼 생산 | 도요타 생산 방식 |
| **Lean** | Lean Manufacturing | 린 생산: 낭비 제거를 통한 효율 극대화 | 도요타, 현대차 |
| **TPM** | Total Productive Maintenance | 전사적 생산 보전: 모든 부서가 참여하는 설비 관리 | 일본 제조업 |
| **Kaizen** | 改善 (개선) | 지속적 개선: 작은 개선을 지속적으로 실행 | 도요타 문화 |
| **Kanban** | 看板 (간판) | 간판 방식: 시각적 신호로 생산량 조절 | JIT의 핵심 도구 |
| **SMED** | Single-Minute Exchange of Die | 1분 이내 금형 교체: 생산 전환 시간 단축 | 다품종 소량 생산 |
| **Poka-Yoke** | ポカヨケ (포카요케) | 실수 방지: 오류를 사전에 방지하는 장치 | 조립 라인 오삽입 방지 |

### 2.2 로봇 및 자동화 용어

| 용어 | 영문 | 정의 | 사양 예시 |
|------|------|------|----------|
| **DOF** | Degrees of Freedom | 자유도: 로봇이 움직일 수 있는 독립적인 축의 수 | 6축 산업용 로봇 (6 DOF) |
| **Payload** | Payload | 가반 하중: 로봇이 들 수 있는 최대 무게 | UR5: 5kg, UR10: 10kg |
| **Reach** | Reach | 작업 반경: 로봇이 도달할 수 있는 최대 거리 | ABB IRB 6700: 3.2m |
| **Repeatability** | Repeatability | 반복 정밀도: 같은 위치로 반복 시 오차 | ±0.03mm (산업용 로봇) |
| **Cycle Time** | Cycle Time | 사이클 타임: 한 작업을 완료하는 시간 | 픽앤플레이스 2초 |
| **Takt Time** | Takt Time | 타크트 타임: 고객 수요에 맞춰 생산해야 하는 시간 간격 | 60초/대 (시간당 60대 생산) |
| **End Effector** | End Effector | 엔드 이펙터: 로봇 팔 끝에 장착되는 도구 | 그리퍼, 용접 토치, 드릴 |
| **Gripper** | Gripper | 그리퍼: 물체를 잡는 로봇 손 | 전기식, 공압식, 진공식 |
| **Cobot** | Collaborative Robot | 협동 로봇: 사람과 함께 일할 수 있는 안전한 로봇 | UR, Doosan, ABB YuMi |
| **AGV** | Automated Guided Vehicle | 무인 운반차: 정해진 경로를 따라 이동하는 차량 | 자기 테이프, QR 코드 유도 |
| **AMR** | Autonomous Mobile Robot | 자율 이동 로봇: 스스로 경로를 찾아 이동하는 로봇 | SLAM, LiDAR 기반 |
| **Teach Pendant** | Teach Pendant | 티칭 펜던트: 로봇을 수동으로 프로그래밍하는 휴대형 장치 | 산업용 로봇 조작기 |
| **IK** | Inverse Kinematics | 역기구학: 목표 위치에서 관절 각도를 계산 | 로봇 경로 계획 핵심 |
| **FK** | Forward Kinematics | 정기구학: 관절 각도에서 엔드 이펙터 위치 계산 | 현재 위치 확인 |
| **SLAM** | Simultaneous Localization And Mapping | 동시 위치 추정 및 지도 작성: 로봇이 스스로 환경 지도를 만들며 위치 파악 | AMR 자율 주행 핵심 |

### 2.3 IoT 및 통신 용어

| 용어 | 영문 | 정의 | 프로토콜 예시 |
|------|------|------|--------------|
| **IoT** | Internet of Things | 사물 인터넷: 센서와 기기를 인터넷으로 연결 | 온도, 진동, 압력 센서 네트워크 |
| **IIoT** | Industrial Internet of Things | 산업 사물 인터넷: 제조업에 특화된 IoT | 공장 설비 모니터링 |
| **Edge Computing** | Edge Computing | 엣지 컴퓨팅: 데이터 수집 지점 근처에서 처리 | 실시간 AI 품질 검사 |
| **Fog Computing** | Fog Computing | 포그 컴퓨팅: 엣지와 클라우드 사이의 중간 계층 | 지역 공장 데이터 집계 |
| **MQTT** | Message Queuing Telemetry Transport | 경량 메시지 프로토콜: IoT 기기 간 통신 | 센서 데이터 전송 |
| **OPC UA** | Open Platform Communications Unified Architecture | 산업 표준 통신 프로토콜: 다른 제조사 설비 간 통신 | Siemens ↔ Rockwell 연동 |
| **Modbus** | Modbus | 산업용 시리얼 통신 프로토콜 (1979년 개발) | PLC ↔ 센서 통신 |
| **CAN Bus** | Controller Area Network | 차량 및 산업용 네트워크 (노이즈 강함) | 자동차 ECU 통신 |
| **EtherCAT** | Ethernet for Control Automation Technology | 고속 실시간 이더넷: 1ms 이하 동기화 | 다축 로봇 제어 |
| **TSN** | Time-Sensitive Networking | 시간 민감 네트워킹: 이더넷에 실시간성 부여 | 5G 산업용 네트워크 |
| **5G URLLC** | Ultra-Reliable Low-Latency Communication | 초고신뢰 저지연 통신: 지연 1ms, 신뢰성 99.999% | 원격 로봇 제어 |
| **DDS** | Data Distribution Service | 실시간 데이터 배포 서비스 | ROS 2 통신 |

### 2.4 AI 및 데이터 용어

| 용어 | 영문 | 정의 | 적용 예시 |
|------|------|------|----------|
| **ML** | Machine Learning | 머신러닝: 데이터로부터 패턴을 학습 | 불량 예측 모델 |
| **DL** | Deep Learning | 딥러닝: 다층 신경망을 이용한 학습 | 이미지 품질 검사 |
| **CNN** | Convolutional Neural Network | 합성곱 신경망: 이미지 인식에 특화 | 용접 비드 검사 |
| **RNN** | Recurrent Neural Network | 순환 신경망: 시계열 데이터 처리 | 센서 데이터 예측 |
| **LSTM** | Long Short-Term Memory | 장단기 메모리: 장기 의존성 학습 | 설비 고장 예측 (3주 전) |
| **GAN** | Generative Adversarial Network | 생성적 적대 신경망: 가짜 데이터 생성 | 불량 데이터 증강 |
| **RL** | Reinforcement Learning | 강화학습: 보상을 통해 최적 행동 학습 | 생산 스케줄 최적화 |
| **CV** | Computer Vision | 컴퓨터 비전: 이미지/영상 분석 | AI 품질 검사, OCR |
| **OCR** | Optical Character Recognition | 광학 문자 인식: 이미지에서 텍자 추출 | 부품 시리얼 번호 읽기 |
| **YOLO** | You Only Look Once | 실시간 객체 탐지 알고리즘 (빠름!) | 컨베이어 물체 검출 (30fps) |
| **R-CNN** | Region-based CNN | 영역 기반 객체 탐지 (정확함) | 정밀 불량 검사 |
| **Transfer Learning** | Transfer Learning | 전이 학습: 기존 모델을 새 작업에 재사용 | ImageNet → 산업 데이터 |
| **Few-shot Learning** | Few-shot Learning | 적은 데이터로 학습: 10개 이하 샘플로 학습 | 희귀 불량 검출 |
| **MLOps** | Machine Learning Operations | ML 운영: AI 모델의 배포, 모니터링, 재학습 자동화 | 모델 성능 지속 관리 |
| **Digital Twin** | Digital Twin | 디지털 트윈: 현실 시스템의 가상 복제본 | 시뮬레이션으로 최적화 |
| **Predictive Maintenance** | Predictive Maintenance | 예측 정비: 고장 전 미리 수리 | 베어링 교체 (2주 전 예측) |
| **Anomaly Detection** | Anomaly Detection | 이상 탐지: 정상에서 벗어난 패턴 찾기 | 설비 이상 신호 감지 |

### 2.5 생산 방식 용어

| 용어 | 영문 | 정의 | 특징 |
|------|------|------|------|
| **Push System** | Push System | 푸시 시스템: 예측에 따라 미리 생산 | 재고 발생 가능, 대량 생산 |
| **Pull System** | Pull System | 풀 시스템: 주문이 들어오면 생산 (JIT) | 재고 최소화, 유연 생산 |
| **MTO** | Make-to-Order | 주문 생산: 주문 후 생산 시작 | 맞춤형 제품 (자동차) |
| **MTS** | Make-to-Stock | 재고 생산: 미리 만들어서 재고로 보관 | 표준 제품 (스마트폰) |
| **ATO** | Assemble-to-Order | 주문 조립: 부품 재고 + 주문 시 조립 | PC, 가구 (Dell 방식) |
| **CTO** | Configure-to-Order | 주문 구성: 고객이 옵션 선택 | 자동차 옵션 선택 |
| **Mass Production** | Mass Production | 대량 생산: 단일 제품 대량 생산 | 컨베이어 벨트 (포드) |
| **Mass Customization** | Mass Customization | 대량 맞춤: 대량 생산 + 맞춤형 | 나이키 신발 커스터마이징 |
| **Batch Production** | Batch Production | 배치 생산: 같은 제품을 묶음으로 생산 | 화학, 식품 산업 |
| **Flow Production** | Flow Production | 흐름 생산: 연속적으로 생산 | 자동차 조립 라인 |
| **Cell Production** | Cell Production | 셀 생산: 소규모 팀이 완제품 조립 | 현대차 HMGICS |
| **Modular Production** | Modular Production | 모듈형 생산: 모듈을 조합하여 제품 완성 | 레고 블록 방식 |

### 2.6 품질 관리 용어

| 용어 | 영문 | 정의 | 목표 |
|------|------|------|------|
| **TQM** | Total Quality Management | 전사적 품질 관리: 모든 부서가 품질 책임 | 불량률 제로 |
| **SPC** | Statistical Process Control | 통계적 공정 관리: 통계로 품질 관리 | 관리도 (Control Chart) |
| **SQC** | Statistical Quality Control | 통계적 품질 관리: 샘플링으로 품질 검사 | 샘플 100개 중 1개 불량 |
| **Six Sigma** | Six Sigma | 식스 시그마: 불량률 3.4 ppm 목표 | 99.99966% 합격률 |
| **ppm** | Parts Per Million | 백만 개당 불량 개수 | 3.4 ppm = 100만 개 중 3.4개 |
| **Cpk** | Process Capability Index | 공정 능력 지수: 공정이 규격 내에서 생산하는 능력 | Cpk ≥ 1.33 (양호) |
| **FMEA** | Failure Mode and Effects Analysis | 고장 모드 영향 분석: 잠재적 고장 사전 분석 | 리스크 평가 |
| **RPN** | Risk Priority Number | 위험 우선순위 수: 발생도 × 심각도 × 검출도 | RPN > 100 시 조치 |
| **FTQ** | First Time Quality | 초회 품질: 첫 번째 생산에서 합격률 | FTQ 99% 목표 |
| **RTY** | Rolled Throughput Yield | 누적 합격률: 전체 공정 합격률 | 95% × 98% × 99% = 92.2% |
| **DPMO** | Defects Per Million Opportunities | 백만 기회당 불량 수 | Six Sigma: 3.4 DPMO |
| **AQL** | Acceptable Quality Level | 합격 품질 수준: 허용 가능한 불량률 | AQL 1.0 (1% 불량 허용) |

### 2.7 생산성 지표 용어

| 용어 | 영문 | 정의 | 계산식 |
|------|------|------|--------|
| **OEE** | Overall Equipment Effectiveness | 종합 설비 효율 | 가동률 × 성능률 × 양품률 |
| **Availability** | Availability | 가동률: 설비가 가동된 시간 비율 | (운전 시간 / 계획 시간) × 100% |
| **Performance** | Performance | 성능률: 실제 생산량 / 이론 생산량 | (실제 생산 / 이상 생산) × 100% |
| **Quality** | Quality | 양품률: 합격품 비율 | (합격품 / 전체 생산) × 100% |
| **MTBF** | Mean Time Between Failures | 평균 고장 간격: 고장 사이의 평균 시간 | 1000시간 (41일) |
| **MTTR** | Mean Time To Repair | 평균 수리 시간: 고장 수리에 걸리는 평균 시간 | 2시간 |
| **Uptime** | Uptime | 가동 시간: 설비가 정상 작동한 시간 | 98% (365일 중 358일) |
| **Downtime** | Downtime | 정지 시간: 설비가 멈춘 시간 | 고장, 교체, 청소, 대기 |
| **Throughput** | Throughput | 처리량: 단위 시간당 생산량 | 100대/시간 |
| **Cycle Time** | Cycle Time | 사이클 시간: 1개 생산에 걸리는 시간 | 36초/대 (테슬라) |
| **Lead Time** | Lead Time | 리드 타임: 주문부터 납품까지 시간 | 3일 (아마존 당일 배송) |
| **WIP** | Work In Progress | 재공품: 생산 중인 반제품 | 최소화 목표 (JIT) |
| **Inventory Turnover** | Inventory Turnover | 재고 회전율: 연간 재고 회전 횟수 | 12회/년 (월 1회) |
