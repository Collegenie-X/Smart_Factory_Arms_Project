# 압력 센서 (FSR) 측정 예제

## 📌 개요
FSR (Force Sensitive Resistor) 압력 센서를 사용하여 압력/힘을 측정하는 예제입니다.

## 🔧 하드웨어
- **FSR 압력 센서**
- **저항** 10kΩ (풀다운)
- **LED** 3개 (압력 단계 표시, PWM)

## 📍 핀 연결
```
Arduino UNO
├─ A0  → FSR 센서 (아날로그 입력)
├─ D3  → 낮은 압력 LED (PWM)
├─ D5  → 중간 압력 LED (PWM)
└─ D6  → 높은 압력 LED (PWM)

FSR 회로:
  5V ─── FSR ─┬─ A0
              │
            10kΩ
              │
             GND
```

## 🎯 동작 원리
1. **압력 감지**: FSR 저항 값 변화
2. **전압 분배**: 압력에 따라 전압 변화
3. **ADC 변환**: 아날로그 값 (0~1023)
4. **LED 제어**: PWM으로 밝기 조절

## 📊 압력 레벨
| 값 | 압력 | LED 상태 |
|------|------|----------|
| 0~200 | 없음 | 모두 OFF |
| 200~500 | 낮음 | LED1 점등 (밝기 가변) |
| 500~800 | 중간 | LED1+LED2 점등 |
| 800~1023 | 높음 | LED1+LED2+LED3 점등 |

## 📝 주요 함수
- `controlLEDs()`: PWM LED 밝기 제어
- `printPressure()`: 압력 값 출력
- `printPressureBar()`: 바 그래프 표시
- `pressureToForce()`: Newton 단위 변환
