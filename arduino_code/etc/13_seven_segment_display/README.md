# 7 Segment 디스플레이 제어 예제

## 📌 개요
7 segment 디스플레이를 사용하여 숫자, 시간, 온도를 표시하는 예제입니다. TM1637 4자리 모듈을 사용합니다.

## 🔧 하드웨어
- **TM1637 4자리 7 Segment 모듈**
- 또는 **단일 7 Segment + 저항**

## 📍 핀 연결 (TM1637)
```
Arduino UNO
├─ D2  → CLK (클럭)
└─ D3  → DIO (데이터)
```

## 📍 핀 연결 (단일 7 Segment)
```
Arduino UNO
├─ D4  → SEG A
├─ D5  → SEG B
├─ D6  → SEG C
├─ D7  → SEG D
├─ D8  → SEG E
├─ D9  → SEG F
├─ D10 → SEG G
└─ D11 → DP (소수점)

주의: 공통 캐소드 기준
각 세그먼트에 220Ω 저항 연결
```

## 🎯 기능
1. **숫자 표시**: 0~9999
2. **시간 표시**: HH:MM (콜론 포함)
3. **온도 표시**: -99~999°C
4. **밝기 조절**: 0~7 레벨
5. **카운트 업**: 0~9999 자동 증가
6. **시계 모드**: 실시간 시간 표시

## 💻 Serial 명령어
| 명령어 | 기능 | 예시 |
|--------|------|------|
| `num:숫자` | 숫자 표시 | `num:1234` |
| `time:HH:MM` | 시간 표시 | `time:12:34` |
| `temp:온도` | 온도 표시 | `temp:25` |
| `bright:레벨` | 밝기 조절 | `bright:5` |
| `count` | 카운트 업 | - |
| `clock` | 시계 모드 | - |
| `clear` | 화면 지우기 | - |

## 📊 7 Segment 구조
```
   A
  ---
F|   |B
  -G-
E|   |C
  ---
   D   DP
```

## 📝 주요 함수
- `displayNumber()`: 숫자 표시 (0~9999)
- `displayTime()`: 시간 표시 (HH:MM)
- `displayTemperature()`: 온도 표시 (°C)
- `setBrightness()`: 밝기 조절 (0~7)
- `countUp()`: 카운트 업
- `clockMode()`: 시계 모드

## 📚 필수 라이브러리
```
TM1637Display
```

## 🔍 응용 분야
- 디지털 시계
- 온도계
- 타이머/카운터
- 점수판
- 계측기

## 💡 TM1637 vs 단일 7 Segment

| 항목 | TM1637 | 단일 7 Segment |
|------|--------|----------------|
| 핀 수 | 2개 (CLK, DIO) | 8개 (A~G, DP) |
| 자리 수 | 4자리 | 1자리 |
| 밝기 조절 | 내장 | 외부 PWM 필요 |
| 가격 | 중간 | 저렴 |
| 사용 편의성 | 쉬움 | 복잡 |
