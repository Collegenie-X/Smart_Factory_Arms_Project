# 8x8 LED Matrix 제어 예제 (MAX7219)

## 📌 개요
MAX7219 드라이버를 사용한 8x8 LED Matrix 제어 예제입니다. 다양한 패턴과 애니메이션을 표시할 수 있습니다.

## 🔧 하드웨어
- **8x8 LED Matrix** (MAX7219 드라이버)
- **5핀 연결**: VCC, GND, DIN, CS, CLK

## 📍 핀 연결
```
Arduino UNO
├─ D11 → DIN (Data In, MOSI)
├─ D13 → CLK (Clock, SCK)
├─ D10 → CS (Chip Select, SS)
├─ 5V  → VCC
└─ GND → GND
```

## 🎯 기능
- 숫자 표시 (0, 1)
- 아이콘 표시 (하트, 웃는 얼굴, 화살표)
- 체크 마크, X 표시
- 자동 애니메이션
- Serial 명령으로 패턴 변경

## 💻 Serial 명령어
| 명령어 | 기능 |
|--------|------|
| `0` | 숫자 0 표시 |
| `1` | 숫자 1 표시 |
| `h` | 하트 표시 |
| `s` | 웃는 얼굴 |
| `u` | 화살표 위 |
| `c` | 체크 마크 |
| `x` | X 표시 |
| `f` | 전체 켜짐 |
| `a` | 자동 애니메이션 |
| `clear` | 화면 지우기 |

## 📊 패턴 정의 방법
```cpp
byte pattern_heart[8] = {
  B00000000,
  B01100110,
  B11111111,
  B11111111,
  B11111111,
  B01111110,
  B00111100,
  B00011000
};
```

## 📝 주요 함수
- `displayPattern()`: 패턴 표시
- `runAnimation()`: 자동 애니메이션 실행
- `handleCommand()`: Serial 명령 처리

## 🔍 응용 분야
- 상태 표시 시스템
- 간단한 게임 (테트리스, 뱀 게임)
- 시계 표시
- 알림 아이콘
