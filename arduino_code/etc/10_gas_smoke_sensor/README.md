# MQ-2 가스/연기 센서 측정 예제

## 📌 개요
MQ-2 가스 센서를 사용하여 가연성 가스 및 연기를 감지하는 예제입니다.

## 🔧 하드웨어
- **MQ-2 가스 센서**
- **LED** 3개 (안전/경고/위험)
- **부저** 1개 (경보음)

## 📍 핀 연결
```
Arduino UNO
├─ A0  → MQ-2 아날로그 출력
├─ D2  → 초록 LED (안전)
├─ D3  → 노랑 LED (경고)
├─ D4  → 빨강 LED (위험)
└─ D5  → 부저 (경보)
```

## 🎯 감지 가스
- LPG
- 프로판
- 메탄
- 수소
- 알코올
- 연기

## ⚠️ 중요 사항
**예열 시간**: 센서 가동 후 약 60초 필요
- 센서 히터가 안정화될 때까지 대기
- 예열 중에는 정확한 측정 불가

## 📊 위험 레벨
| 값 | 레벨 | LED | 부저 |
|------|------|-----|------|
| 0~300 | 안전 | 초록 | OFF |
| 300~600 | 경고 | 노랑 | 느린 비프음 |
| > 600 | 위험 | 빨강 | 빠른 비프음 |

## 📝 주요 함수
- `checkWarmupStatus()`: 예열 상태 체크
- `controlAlarm()`: 경보 제어
- `calculatePPM()`: PPM 단위 변환
- `calibrateSensor()`: 센서 캘리브레이션

## 🔧 캘리브레이션
```cpp
// 깨끗한 공기에서 기준값 측정
int baseline = calibrateSensor();
```

## 🔍 응용 분야
- 가스 누출 감지
- 화재 조기 경보
- 주방 환기 자동화
- 산업 안전 시스템

## ⚠️ 주의 사항
- 실제 안전 장비 대용으로 사용 금지
- 정기적인 캘리브레이션 필요
- 센서 수명 약 3~5년
